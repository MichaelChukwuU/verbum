e-Library


An Electronic Library (E-Library) is a Physical site and/or website that provides online access to digitized audio, video, and written material stored as electronic media formats (as opposed to print, or other media), along with means for organizing, storing, and retrieving the files and media contained in the library collection.

E-libraries can vary immensely in size and scope, and can be maintained by individuals, organizations, or affiliate d with established physical library buildings or institutions, or with academic institutions.

E-libraries can either be locally hosted within the school or organization’s premises and books retrieved via an intranet, or it can be hosted on the web which allows access of the books through the internet.

To run an electronic library, one would need the hardware components as well as software components. The hardware components are the computer systems with a local network to connect them with each other as well as internet connectivity to connect the network to the world-wide web. The software components are the e-library interface or portal with access to various online libraries that provide open access, location access or subscription-based access.
